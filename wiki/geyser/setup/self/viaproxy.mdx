---
title: Geyser ViaProxy 搭建
description: 学会配置 Geyser.
---

1. [安装并配置 ViaProxy](https://github.com/ViaVersion/ViaProxy?tab=readme-ov-file#usage-for-players-gui).
2. [点击下载合适的 Geyser 版本](/download).
3. 将 Geyser 文件放入 `plugins` 文件夹, 然后重启服务器.
4. 打开 Geyser 配置文件夹内的 `config.yml`, 你将会看到如下内容:

    ```yaml
    bedrock:
      # 允许访问的 IP 地址
      # 通常来说, 你不应该取消注释, 除非你要限制只有来自某个 IP 地址的访问被允许.
      #address: 0.0.0.0

      # 基岩版 服务器端口.
      port: 19132
    ```
    你只需要根据注释配置它们.
    详见[理解配置](/wiki/geyser/understanding-the-config/) 获取更更多信息.

    如果你使用 ViaProxy 图形化程序, 你可以使用它的账号管理系统让基岩版玩家加入在线验证服务器. 如果你使用 ViaProxy 命令行版本, 那么你需要 [下载并使用 Floodgate](/wiki/floodgate/setup). <br/>
    注意: `auth-type` 设置项将会失效, 并由 ViaProxy 自动处理.

5. 连接到你的服务器:

   **本地连接**
   在同台主机上, 你可以使用 `localhost`, 或者是 `127.0.0.1` 作为连接地址.
   你可能遇到环回限制(Loopback restrictions), 详见[此处修复](/wiki/geyser/fixing-unable-to-connect-to-world#using-geyser-on-the-same-computer).
   在同一子网下的其他设备可以使用你的本地 IPv4 作为连接地址.
   或者你可以通过基岩版客户端中的 好友 选项卡加入服务器.

   **远程连接**
   就像让你的 Java 服务器可以被连接一样进行操作.
   不同的是, 你必须保证 Geyser 端口必须使用 UDP 协议.

6. 使用 `geyser connectiontest <ip> <port>` 命令验证是否能够连接.

:::info
如果想要基岩版玩家不使用正版账户登录, 请使用 <a href="/wiki/floodgate/setup/">Floodgate</a>.
:::
