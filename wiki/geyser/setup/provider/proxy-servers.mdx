:::info
- 请仅在代理服务端上安装 Geyser! Floodgate 允许并建议安装在所有子服务器.
- 如果基岩版玩家想要进入一个子服, 那么这个子服必须允许 <Versions platform="java"/> Java 客户端.
:::

1. 选择你的面板服务商查看相关事项.
2. [点击下载合适的 Geyser 版本](/download).
3. 将 Geyser 文件放入 `plugins` 文件夹, 然后重启服务器.
4. 打开 Geyser 配置文件夹内的 `config.yml`, 你将会看到如下内容:

    ```yaml
    bedrock:
      # 允许访问的 IP 地址
      # 通常来说, 你不应该取消注释, 除非你要限制只有来自某个 IP 地址的访问被允许.
      #address: 0.0.0.0

      # 基岩版 服务器端口.
      port: 19132

      # 该选项使每次启动服务器时, 基岩版服务器端口与 Java 版服务器端口相同.
      clone-remote-port: false
    ``` 
    你只需要根据注释配置它们.
    详见[理解配置](/wiki/geyser/understanding-the-config/) 获取更更多信息.

5. 使用 `geyser connectiontest <ip> <port>` 命令验证是否能够连接.

:::info
如果想要基岩版玩家不使用正版账户登录, 请使用 <a href="/wiki/floodgate/setup/">Floodgate</a>.
:::
